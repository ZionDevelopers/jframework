jFramework
==========

This is a PHP POO MVC framework, have built-in:

* Database Manager
* Image Handler
* Captcha Generator
* Cielo _(Brazilian Credit Card Gateway)_
* BoletoPhp (Brazilian Billet Generator)
* PHPMAILER
* Automatic Session Cleaner (Cronjob)
* Tools (tons of useful functions)
